How to code in any programming language

  1. Choose an easy programming language first
    1. Python and Swift are excellent for beginners
  2. Familiarize yourself with the language syntax
    1. Learning the syntax and structure of the language will help you understand how everything comes together
  3. Make simple print programs, for loops, etc
    1. This should also help acquaint you with the syntax as well
  4. Learn about data structures and how they work in code
    1. Data structures are algorithms that can help make your code much more efficient
  5. Stick with it and challenge yourself!
    1. Practice makes perfect!
